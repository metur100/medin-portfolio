Awalon - CRM System

Comprehensive Customer Relationship Management System with SharePoint Integration

Awalon is a sophisticated CRM system designed to manage customer relationships, offers, and business processes. The platform integrates with SharePoint to provide seamless data management and workflow automation for businesses.

Key Features:

  • Responsive Design: Adapts to desktops, tablets, and mobile devices, providing seamless user experience.
  • Customer Management: Create and manage customer profiles, companies, and contact information.
  • Offer Management: Create, track, and manage customer offers with detailed information.
  • SharePoint Integration: Seamless integration with Microsoft SharePoint for data storage and management.
  • Reporting & Analytics: Comprehensive reporting features with classification and portfolio management.

Technical Specifications:

  • Frontend Technologies: Next.js, React, modern JavaScript, responsive CSS.
  • Backend Technologies: C# .NET for robust API development and business logic.
  • Data Integration: Microsoft SharePoint integration for secure data management.
  • Security: Secure authentication and authorization mechanisms.

Core Functionalities:

  • Company Management: Create and organize company profiles with detailed information.
  • Contact Management: Maintain comprehensive contact databases with relationship tracking.
  • Portfolio Classification: Categorize and manage customer portfolios with customizable criteria.
  • Commission Tracking: Monitor and calculate commissions and success rates.
  • Advanced Search: Powerful search functionality to quickly find customers, offers, and documents.
