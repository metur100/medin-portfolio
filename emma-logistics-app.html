Emma Logistics Platform

Comprehensive Freight and Vehicle Management for the Modern Logistics Industry

The Emma Logistics Platform is a next-generation solution designed to streamline freight and vehicle management. Built to enhance operational efficiency, this platform provides logistics professionals with real-time tracking, secure vehicle assignment, and seamless shipment coordination. The Emma Logistics ecosystem integrates both web and mobile applications, allowing for continuous management on the go.

Platform Highlights:

  • Vehicle and Freight Management: Assign vehicles to shipments with detailed specifications and real-time availability.
  • Email Notifications: Receive alerts for shipment milestones, delivery statuses, and critical updates.
  • Secure API Integration: Utilize OAuth 2.0 and JWT for secure communication between services and users.
  • Multilingual Support: Full language localization using i18next with primary support for Bosnian.
  • User Authentication: Email and password-based login with basic authentication and JWT token management.

Technical Specifications:

  • Frontend: React (Vite) + TypeScript for a fast, responsive interface.
  • Backend: ASP.NET Core 8 (RESTful API architecture).
  • API Calls: Axios for efficient, secure API communication.
  • Authentication: OAuth 2.0, JWT, and Basic Auth with user credentials (email/password).
  • Routing: Dynamic HTML and CSS link routing for seamless navigation.
  • Data Handling: MS SQL DB for structured data storage and retrieval.
